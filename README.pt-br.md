# Sistema de Biblioteca Fictícea

Um sistema de biblioteca simples que pode cadastrar até 20 livros em um acervo.

Each book can contain the following information:

* Código.
* Titulo.
* Autor.
* Área.
* Editora.
* Ano de Publicação.

O sistema permite que o usuário:

* Pesquise um livro pelo seu código.
* Imprima todos os livros cadastrados e suas informações.
* Ordene todos os livros cadastrados em ordem crescente baseado no seu ano de publicação.
